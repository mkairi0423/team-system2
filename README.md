***********    Gitに自分のソースコードを追加　   **********************

１．今いるブランチの表示
git branch
例：feature/backend ,feature/frontenが出てくると思う


２．ブランチに自分が書いたソースコードを追加 
※一行ずつやる

git add .
git commit -m "コメント"
                ↑ここはなんでもいい　例：作っている画面の名前だったり、追加したコードの機能だったり　
                とりあえず抽象的な名前は良くない

git push origin /*自分のブランチを書く*/
                例：feature/backend , feature/frontend

３．ブラウザでGitを開きpullリクエストをする
